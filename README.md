## 🚀 Overview
Ever wanted to save a brilliant conversation you had with an AI, but screenshots were too messy and copy-pasting ruined the formatting? **ChatPDF.io** solves this gracefully. 
By utilizing advanced stealth headless-browser scraping, this tool bypasses complex bot protections and perfectly converts your shared AI links into luxurious, print-ready PDF files—while retaining full formatting including Markdown, code blocks, and tables.

## ✨ Key Features
- **🔥 One-Click Extraction:** Just paste a shared link from ChatGPT, Claude, or Gemini.
- **👻 Stealth Security Bypass:** Uses `puppeteer-extra-plugin-stealth` to seamlessly bypass Cloudflare Turnstile barriers and HTTP 403 errors.
- **🎨 Elite UI/UX:** Features a state-of-the-art landing page with fluid 3D animations, aesthetic spatial typography, and responsive Web3 grid designs.
- **📖 Beautiful PDF Maker:** Automatically injects the premium `Plus Jakarta Sans` font and applies custom styling overheads to the final exported PDF.
- **⚡ Super Smart Handler:** Handles complex SPA (Single Page Application) hydration issues, delays, re-routes, and automatically clicks application-retry verification buttons.

## 🛠️ Tech Stack
- **Frontend Layer:** HTML5, CSS3 (Floating 3D Animations & Gradients), Modern Vanilla JS
- **Backend API:** Node.js, Express.js
- **Scraping Engine:** Puppeteer (Headless Browser) + Stealth Plugins
